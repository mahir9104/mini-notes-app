# 📝 Notes App

A simple notes app built with React and Tailwind CSS.

## Features
- Add notes with title and detail
- Delete notes
- Responsive design

## Tech Stack
- React
- Tailwind CSS
- Vite
- Lucide React

## Live Demo
[View Live](https://mini-notes-app-ivory.vercel.app)

## Getting Started
```bash
npm install --legacy-peer-deps
npm run dev
```
